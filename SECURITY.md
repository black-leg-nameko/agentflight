# Security policy

AgentFlight records commands, model interactions, and workspace changes, so security reports may involve sensitive local data.

Please do not open a public issue for a suspected vulnerability. Use GitHub's private vulnerability reporting for this repository and include the affected version, reproduction steps, impact, and any suggested mitigation. Do not attach unredacted `.afrun` bundles or real credentials.

The current `main` branch receives security fixes. AgentFlight cannot guarantee complete secret detection, complete sandboxing, or restoration of external service state; review exported bundles before sharing them.
