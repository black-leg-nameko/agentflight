# Contributing

Thanks for helping build AgentFlight.

1. Open an issue for substantial behavior or schema changes before implementation.
2. Branch from `main` and keep each pull request focused.
3. Run `cargo fmt --all -- --check`, `cargo clippy --workspace --all-targets -- -D warnings`, and `cargo test --workspace --locked`.
4. Include migrations and compatibility tests with storage or public-schema changes.
5. Never commit credentials, private Run bundles, or customer data.
